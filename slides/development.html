Les extensions dans TYPO3

Spend one hour and it works one place - spend two hours and it works in a hundred places
  • Rapidité de partage : upload / download
  • Rapidité de développement : extension_builder (kickstart) / modularité / point d'entrée limité
  • Facilité de documentation : intégrée aux extensions / reStructuredText (pdf, html ...)
  • Séparation des responsabilités : chaque développeur est responsable du code au sein de ses extensions
  • Point de gestion unique : gestionnaire d'extensions (extension_manager)

Clés d'extensions :

dossier + namespaces


preg_match('/[a-z0-9_]{3,30}/', $extensionKey)
&& !preg_match('/^(tx|user_|pages|tt_|sys_|ts_language_|csh)/', $extensionKey)
&& !preg_match('/(tx|user_|pages|tt_|sys_|ts_language_|csh)$/', $extensionKey)
&&!preg_match('/^[0-9_]+/', $extensionKey)
&&!preg_match('/_[0-9]/', $extensionKey)
